# Introduction to the Bulk Data Migration Tool

Prepare Docker and access the Bulk Data Migration Tool to migrate Adobe Commerce data to [!DNL Adobe Commerce as a Cloud Service] ([!DNL ACCS]).

## Who is this video for?

* Solutions Architect
* DevOps Engineer
* Backend Developer

## Video content

* Learn the capabilities and limitations of the Bulk Data Migration Tool.
* Confirm Docker is installed and the Docker engine is running before you start.
* Extract the tool package and set the correct folder permissions.
* Review the `.env.example` file configured in the next videos of this series.
