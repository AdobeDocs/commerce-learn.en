# Create an App Builder project

Learn how to create an Adobe Developer App Builder project to consume events from Adobe Commerce. Additional documentation found at [Install Adobe I/O Events for Adobe Commerce](https://developer.adobe.com/commerce/extensibility/events/installation){target="_blank"}.

## Who is this video for?

* Developers new to Adobe Commerce and Adobe Developer App Builder using I/O events who need to create an Adobe App Builder project.

## Video content {#video-content}

* Creating an App Builder project
* Using Adobe Developer Console
* Using Adobe I/O

>[!VIDEO](https://video.tv.adobe.com/v/3415797?learn=on)

## Useful commands {#useful-commands}

```bash
mkdir myproject && cd myproject

aio login

aio app init

aio app run
```
