# Use the Upgrade Compatibility Tool in PhpStorm

{{only-for-on-prem-commerce-cloud}}

The Upgrade Compatibility Tool (UCT) analyzes incompatibilities between your current version and a target upgrade version in minutes. The PhpStorm plugin makes the tool easier to run from your IDE.

In this video:

* What the Upgrade Compatibility Tool does
* How to use it with the Magento PhpStorm plugin
* How to read HTML results by module severity and incompatibility count

## Who is this video for?

* Developers

## Video content

* Install and run UCT from PhpStorm.
* Review HTML reports and prioritize fixes.
