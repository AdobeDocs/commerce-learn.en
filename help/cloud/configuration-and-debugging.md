# Configuration and debugging

This video is part of the Adobe Commerce Cloud technical onboarding series. It explains how YAML configuration files work across environments, which tools help you validate and optimize them, and how you approach debugging and logging. You also see how ECE-Tools and the Quality Patches Tool fit in, how New Relic aggregates logs, how to use Xdebug for remote debugging, and how to manage logs across multiple application servers.

## Who is this video for?

* DevOps engineers
* Commerce architects
* Backend developers

## Video content

* YAML configuration across cloud environments and environment-specific settings.
* Validation, optimization, and debugging workflows.
* ECE-Tools, Quality Patches Tool, New Relic, Xdebug, and log management.
